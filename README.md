# Asset-Management-GUI-Work
Asset Management System built with Java, JavaFX, and MySQL for tracking inventory, user assignments, and maintenance records.
A full-stack desktop Asset Management System built using Java, JavaFX (Scene Builder), and MySQL.
This application allows users to manage inventory, assign assets to employees, track maintenance history, and monitor asset availability through an interactive UI.

Designed as a portfolio project to showcase full-stack development and database-driven application design.
